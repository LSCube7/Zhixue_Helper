# 更新日志

## v2.2.1 - 2026-07-15

### 修复

- 精简扩展权限声明，移除未使用的本地存储、标签页临时访问及多余主机访问权限，解决 Chrome Web Store 的过度权限审核问题。

### 优化

- 扩展现在仅查询已声明的智学网页面，同时保留连接恢复、作业资源下载和必要网络请求能力。

### 注意事项

- 本次更新不会清除主题设置、成绩缓存或分析配置，也不需要用户重新登录或重新授权。

## v2.2.0 - 2026-07-12

### 新增

- 新增公开演示模式，无需登录智学网即可体验考试列表、考试详情、成绩趋势、分类规则和作业资源。
- 演示模式内置两学年、十二场考试、六科成绩和虚拟作业；进入后会直接展示六场考试的分析结果。
- 演示作业支持预览和下载扩展内置的示例文本、图片，不会请求真实作业资源。

### 优化

- 演示数据中的语文、数学、英语、物理、化学、生物分别围绕 105、125、130、70、85、80 分波动。
- 加大理科成绩波动、减小文科成绩波动，并在多场考试中展示成绩或排名异常，方便体验趋势分析。
- 演示方案、模板、备注、分类规则和分析阈值仅在当前页面中生效，与用户真实 IndexedDB 缓存完全隔离。
- 退出演示后自动恢复真实连接和本地数据；重新打开 Owl Insight 时默认进入真实模式。

### 修复

- 修复 HCT 颜色选择和作业资源预览对话框的遮罩无法覆盖整个浏览器窗口的问题。
- 修复 Pride Color 面板被后续分析设置卡片覆盖的问题。
- 修复作业列表进入箭头换行显示的问题，箭头现在始终固定在卡片右侧。

### 注意事项

- 本版本不新增浏览器权限，也不会在演示模式中访问智学网、读取 Cookie 或写入真实成绩缓存。
- 更新后请在扩展管理页重新加载 Owl Insight；已有缓存、主题和分析配置不会丢失。

## v2.1.0 - 2026-07-12

### 新增

- 新增独立连接页，可选择已登录的智学网页面，并展示学生姓名、登录名、头像、学校、年级与班级资料。
- 新增作业资源页，支持已提交/未提交筛选、科目筛选、分页加载及题目、答案、提交、说明资源解析。
- 作业支持独立详情页、文本与图片预览、单项下载、资源多选和批量下载。
- Manifest 增加中文与英文说明，扩展名称统一为 Owl Insight。
- 单次考试详情加入多科雷达图。

### 优化

- `xToken` 改为通过当前智学网 Cookie 会话临时获取，不再从 localStorage 读取或持久化。
- 作业接口改由扩展后台处理，并使用限定到扩展自身与作业域名的网络规则解决跨域预检。
- 在线考试与缓存成绩统一使用考试卡片布局，缓存考试可以直接打开详情并参与离线分析。
- 综合分析移除单科成绩列和雷达图，改为从分析结果直接打开对应考试。
- 成绩异常默认收起，并按考试、科目及对比考试精确说明变化。
- 分析结果加入“关闭分析”，保留考试选择、方案、模板、目标线与备注。
- 全局提示改为符合 Material Design 3 的底部 Snackbar，并在 6 秒后自动收起。
- 页脚使用 Owl Insight SVG Logo，补充动态主题与无障碍状态反馈。

### 修复

- 修复作业后台消息接收、会话验证及跨域请求初始化失败的问题。
- 修复作业资源复选框因事件对象失效导致的崩溃。
- 修复作业卡片在紧凑窗口中隐藏或错位的进入箭头，箭头始终位于卡片右侧。
- 修复图片或文本资源缺少页内预览入口的问题。

### 注意事项

- 新版本新增 `downloads`、`scripting`、`declarativeNetRequest` 及 `https://mhw.zhixue.com/*` 权限，用于作业资源获取、下载和连接恢复；未申请 `cookies` 权限。
- 更新扩展后需要重新加载扩展并刷新已登录的智学网页面；浏览器可能新增“管理下载”和“管理网络请求”权限提示。
- Owl Insight 不会把 Cookie 或临时 Token 写入 localStorage、IndexedDB 或日志，也不会尝试解锁服务端未返回的答案资源。

## v2.0.0 - 2026-07-08

### 新增

- 迁移到 Vite、React、TypeScript 与 CRXJS 的 Manifest V3 架构。
- 加入考试缓存、趋势分析、分析模板、考试分类、目标线和考试备注。
- 加入 Material Design 3 动态主题、导航栏和本地优先的数据管理体验。

### 优化

- 使用 IndexedDB 保存成绩缓存与分析配置。
- 支持多学年、考试类型、分类规则和缓存成绩参与分析。
- 精简 Material Symbols 字体资源并提供 v2.0.0 扩展安装包。
